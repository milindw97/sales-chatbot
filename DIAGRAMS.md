## Data Flow - Ingestion

```
┌─────────┐
│  START  │
└────┬────┘
     │
     ▼
┌──────────────────┐
│ Read .txt file   │
└────┬─────────────┘
     │
     ▼
┌──────────────────────────┐
│ Extract call_id          │
│ (from filename)          │
└────┬─────────────────────┘
     │
     ▼
┌──────────────────────────┐
│ Parse & create chunks    │
│ - Size: 512 chars        │
│ - Overlap: 50 chars      │
│ - Extract timestamps     │
└────┬─────────────────────┘
     │
     ▼
┌──────────────────────────┐
│ Generate embeddings      │
│ (batch processing)       │
└────┬─────────────────────┘
     │
     ├────────────────┬─────────────┐
     │                │             │
     ▼                ▼             ▼
┌─────────┐    ┌──────────┐   ┌──────────┐
│ Add to  │    │  Store   │   │  Save    │
│  FAISS  │    │ metadata │   │  FAISS   │
│         │    │ in SQLite│   │  index   │
└─────────┘    └──────────┘   └──────────┘
     │
     ▼
┌─────────┐
│   DONE  │
└─────────┘
```

## Data Flow - Query (RAG)

```
┌─────────────┐
│ User Query  │
└──────┬──────┘
       │
       ▼
┌──────────────────┐
│ Rewrite query    │
│ (if history > 0) │
└──────┬───────────┘
       │
       ▼
┌──────────────────┐
│ Embed query      │
└──────┬───────────┘
       │
       ▼
┌──────────────────────┐
│ FAISS similarity     │
│ search (top-k)       │
└──────┬───────────────┘
       │
       ▼
┌──────────────────────┐
│ Get FAISS indices    │
│ [7, 12, 3, 9, 1]     │
└──────┬───────────────┘
       │
       ▼
┌──────────────────────┐
│ Lookup chunks in DB  │
│ (text + metadata)    │
└──────┬───────────────┘
       │
       ▼
┌──────────────────────────────┐
│ Build context with sources   │
│ [Source 1 - call_x [00:05]]  │
│ chunk text...                │
│ [Source 2 - call_y [02:30]]  │
│ chunk text...                │
└──────┬───────────────────────┘
       │
       ▼
┌──────────────────────────────┐
│ Create LLM prompt            │
│ System: You are an analyst   │
│ User: Context + Question     │
└──────┬───────────────────────┘
       │
       ▼
┌──────────────────────┐
│ Call LLM             │
│ (Ollama or Gemini)   │
└──────┬───────────────┘
       │
       ▼
┌──────────────────────────────┐
│ Format response              │
│ - Answer text                │
│ - Source citations           │
│ - Similarity scores          │
└──────┬───────────────────────┘
       │
       ▼
┌──────────────┐
│ Return to    │
│ User         │
└──────────────┘
```
